## Plan: Streamlit → Custom Frontend Migratie

Migreer de Streamlit-app naar een professionele **React + FastAPI** architectuur in de huisstijl van Studio Nico Wissing. De bestaande core backend-logica blijft intact; we wikkelen er een FastAPI REST API omheen en bouwen een React/TypeScript frontend.

### Fase 1: FastAPI Backend *(blokkeert Fase 2)*

1. **API skeleton** — `api/` folder met FastAPI app, CORS, routers
2. **Auth** (`POST /api/auth/login` → JWT token) — hergebruik hmac-check uit auth.py, voeg `python-jose` toe
3. **Generate** (`POST /api/generate`) — wrap `build_all_outputs()` + `export_dxf()` uit pipeline.py in Pydantic-gevalideerd endpoint
4. **Preview** (`POST /api/preview`) — wrap `preview_image()`, return PNG als StreamingResponse
5. **Files** (`GET /api/files/{job_id}`, `/zip`) — bestandslijst + download, hergebruik output folder structuur
6. **Quickscan** (`POST /api/quickscan/{job_id}`) — extract OpenAI vision logic uit main.py naar eigen module
7. **Geocoding** (`GET /api/geocode/suggest?q=...`) — wrap `address_to_rd()` uit locatie.py voor autocomplete
8. **Health + Cleanup** — `/health` endpoint, cleanup scheduler als FastAPI lifespan event

**Nieuwe deps**: `fastapi`, `uvicorn[standard]`, `python-jose[cryptography]`, `python-multipart`

---

### Fase 2: React Frontend *(parallel met Fase 1 tests)*

1. **Setup** — `frontend/` met Vite + React + TS, Tailwind CSS, design tokens als CSS vars, Google Fonts
2. **Layout** — Header (logo + nav + user menu), content area, footer (© Studio Nico Wissing)
3. **Login pagina** — Minimalistisch formulier, logo erboven, JWT opslaan, AuthContext provider
4. **Generator pagina** — Toggle Adres/RD, autocomplete (PDOK suggest), radius slider, preview knop, generate knop (groen, primary), loading spinner
5. **Resultaten** — Bestanden als cards (DXF prominent, PNGs als thumbnails), ZIP download, individuele downloads
6. **Quickscan** — Secties als accordion/tabbladen, kaartafbeelding + AI-analyse tekst
7. **API client layer** — axios instance, JWT interceptor, typed API calls
8. **Responsive** — Mobile-first, sidebar → drawer op mobile

---

### Fase 3: Integratie & Deployment

1. **Dockerfile** — Multi-stage build (Node → build React, Python → FastAPI + deps, kopieer build output naar `/app/static/`)
2. **Nginx** — `/api/` → FastAPI, `/` → static React, WebSocket config verwijderen, timeout 600s behouden
3. **Docker Compose** — Port 8000 (uvicorn) i.p.v. 8501, `JWT_SECRET_KEY` env var toevoegen
4. **Cleanup** — Streamlit uit requirements.txt verwijderen, main.py deprecated/verwijderd
